Features

  • Easily configurable by changing options
  • No external dependencies, apart from jQuery
  • CentipedeJS does not contain a lot of functionality and it keeps the size small, only 30kb

Browser support

  • Firefox
  • Chrome
  • Internet Explorer 9+
  • Opera

How to use

CentipedeJS is a jQuery-plugin for slide show that allows to switch the position of sidebar. Thus, sidebar can be located from left, right and bottom sides of a main slider. That's all :)
In case you want to make some changes then the options are available below.

Download

Download and unpack zip file from github repository.

Installation

  • Include the CSS at the top of your page in your tag:
  •                         
                            <link href="path/to/centipede.css" rel="stylesheet">
  • Make sure to include jQuery in your page:
  •                             
                                <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>
  • Include the Javascript at the bottom of your page before the closing < /body> tag:
  •                             
                                <script src="path/to/centipede.js">< /script>
  • Write your HTML:

Options

                    $(' #my_gallery ').centipedejs({
                        

                            
    
                                
  • position :   "bottom", 
    • 
                                
  • caption :   false, 
    • 
                            

                        

                        });

Option

Default

Description
position: "bottom" Allows you to set position of sidebar, by default it is set to bottom, but you can position sidebar from left or right sides of a slider
caption: false It is possible to add a caption. Therefore caption should be set to true then div with a c_caption class should be added below each image placed inside c_thumbnails unordered list.
8

Change sidebar's position

The location of sidebar can easily be customized by changing it's position in options.

Set captions

All you need for captions is add <div  class="c_caption"> Caption Label </div> div below images.
